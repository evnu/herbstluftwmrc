# herbstluftwm

Configuration for [herbstluftwm](https://herbstluftwm.org).

## Requirements

* urxvt
* dzen2
    * Configuration of colors and font via Xresources
* dmenu: used in some example scripts
* conky
    * Configuration with ~/.conkyrc

## Installation

```
ln -s $PWD ~/.config/herbstluftwm
```
